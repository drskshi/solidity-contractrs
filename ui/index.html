RentalAgreement DApp

Testing only — not real money. Use Hardhat localhost (or Remix VM). ETH is unlimited fake ETH for learning. Do not connect this school demo to Ethereum mainnet or send real funds.

Student tip (MetaMask): On localhost it may show a dollar “fee” or Review alert / HTTP warning. That is normal. You pay gas in fake ETH only (your Hardhat account already has plenty). Tap Review alert → read it → then Confirm. You are not charged real money on chain ID 31337.

Default amounts below are tiny on purpose (e.g. 0.001 ETH rent). Customize rent, deposit, and lease length; the Activity log shows every step—no console required.

1 · Contract & wallet

Do not paste 0xf39F…92266 or your MetaMask address — that is a wallet. Wrong address causes could not decode result data.

Not connected.

Deploy the contract, paste its address, then connect.

2 · Custom numbers (create agreement)

For grading demos, keep small test numbers on localhost only. Values are in ETH on this screen; the contract stores wei. Rent and deposit must stay identical when you pay later.

3 · Two easy buttons (switch MetaMask account between them)

The contract requires the tenant to exist on-chain first. Use the buttons in this order:

  1. Tenant wallet → Tenant: register · sign · deposit (first time: only registers; click again after step 2).
  2. Landlord wallet → Landlord: register · create agreement.
  3. Tenant wallet → same Tenant button again → signs and sends deposit automatically.
  4. (Optional) Pay rent · Skip time · Landlord withdraw — see section 4.

Deposit sent uses the exact amount stored on-chain (from create agreement) so it always matches.

4 · Optional extras

After deposit: tenant can pay rent. After lease ends (or use Skip time below), landlord can withdraw the deposit.

Withdraw: landlord, after endDate, unless lease ended early (then tenant withdraws). Use Hardhat time skip for demos.

5 · Local testing · skip time (Hardhat only)

If MetaMask points to http://127.0.0.1:8545 (run npx hardhat node in the project), you can mine extra time without waiting.
